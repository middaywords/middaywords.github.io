* 使用示例 `is_enum`

  ```c++
  #include <iostream>
  #include <type_traits>
   
  class A {};
   
  enum E {};
   
  enum class Ec : int {};

      std::cout << std::boolalpha;

  }
  ```

* 继承构造函数

  ```c++
  struct Base {
    void f(double i) { std::cout << i << std::endl; }
  }

    // inherits base object constructor here.
    using Base::f;
    void f(int i) { std::cout << i << std::endl; }

    b.f(4.5);

    d.f(4.5);
  }
  ```

* 模板参数可作为友元

  ```c++
  template <typename T> class ClassA {
    public:
    	friend T;
    	void funcA() {}
    private:
    	void funcAprivate() {}
  }
  
  using ClassATest = ClassA<int>;
  
  // 这样在测试的时候就方便一些
  #ifdef UNIT_TEST
  class UtClass {
    public:
    	void ValidateA(int x, ClassATest& a);
  }
  #endif
  ```

  这样就可以任意指定友元了

* Final/override 控制

  override ：编译期会检查该函数是否为重载函数

  final：重载，且该函数不可被子类重载。

  ```c++
  class ClassA : public BaseClass {
    void func1() override;
    void func2() final;
  }
  ```

  `func1()` 和 `func2()` 都是重载`BaseClass`的函数，但 `func2()` 不可被`ClassA`的子类重载

* 模板参数的默认参数

  ```c++
  template <typename T = int>
  	class ClassA {};
  
  template <typename T = int>
  	void func {}
  ```

- nullptr & NULL : 解除了NULL和0的二义性

    ```c++
    void F(int a){
        cout<<a<<endl;
    }

    void F(int *p){
        assert(p != NULL);
        cout<< p <<endl;
    }

    int main(){
        int *p = nullptr;
        int *q = NULL;
        
        bool equal = ( p == q ); 
        // equal的值为true，说明p和q都是空指针
        
        int a = nullptr; 
        // 编译失败，nullptr不能转型为int
        
        F(0); 
        // 在C++98中编译失败，有二义性；在C++11中调用F（int）
        
        F(nullptr);

        return 0;
    }
    ```

- lambda function

    ```c++
    vector<int> iv{5, 4, 3, 2, 1};
    int a = 2, b = 1;

    for_each(iv.begin(), iv.end(), [b](int &x){cout<<(x + b)<<endl;}); // (1)

    for_each(iv.begin(), iv.end(), [=](int &x){x *= (a + b);});		// (2)

    for_each(iv.begin(), iv.end(), [=](int &x)->int{return x * (a + b);});// (3)
    ```

    > 1. []内的参数指的是Lambda表达式可以取得的全局变量。(1)函数中的b就是指函数可以得到在Lambda表达式外的全局变量，如果在[]中传入=的话，即是可以取得所有的外部变量，如（2）和（3）Lambda表达式
    > 2. ()内的参数是每次调用函数时传入的参数。
    > 3. ->后加上的是Lambda表达式返回值的类型，如（3）中返回了一个int类型的变量
